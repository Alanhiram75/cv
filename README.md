# Alan Hiram | Portfolio / CV

Bienvenido al repositorio de mi portafolio personal y currículum vitae interactivo. Este sitio web fue diseñado con una arquitectura moderna estilo "Bento Grid" para mostrar de manera estructurada y visual mis habilidades, experiencia y proyectos.

🌍 **Sitio Web en Vivo:** [https://alanhiram75.github.io/cv/](https://alanhiram75.github.io/cv/)

## 🚀 Características

- **Diseño "Bento Grid":** Interfaz limpia, responsiva y fácil de leer en cualquier dispositivo (móvil, tablet, escritorio).
- **Soporte Multilingüe (i18n):** El sitio cuenta con un sistema de traducción dinámico hecho a medida con Vanilla JavaScript, permitiendo alternar instantáneamente entre Español e Inglés.
- **Modo Oscuro Elegante:** Paleta de colores cuidadosamente seleccionada para un contraste alto, lectura cómoda y estética premium.
- **Rendimiento Optimizado:** Construido sin frameworks pesados para garantizar una carga ultrarrápida.

## 🛠️ Tecnologías Utilizadas

Este proyecto demuestra un dominio sólido sobre las tecnologías web fundamentales:

- **HTML5:** Estructura completamente semántica y accesible.
- **CSS3:** Estilos avanzados utilizando CSS Grid y Flexbox, variables personalizadas (Custom Properties), y efectos de "hover" sutiles.
- **JavaScript (ES6):** Manejo de eventos del DOM y sistema de internacionalización basado en módulos (`export/import` con `data.js`).
